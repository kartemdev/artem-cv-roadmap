# Artem CV Roadmap
Учебный репозиторий для изучения Machine Learning / Computer Vision с использованием Python.

## Цель
Научиться строить полный CV-pipeline:

1. Анализ датасета
2. Предобработка изображений
3. Классическое Computer Vision
4. ML baseline
5. PyTorch classification
6. Detection / segmentation
7. Inference API

## Установка
```bash
uv venv
source .venv/bin/activate
uv sync
```
